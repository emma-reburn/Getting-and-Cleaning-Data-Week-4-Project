# Description of Dataset

Tidydata.txt contains 68 columns with 180 observations. Each row is the average of each variables for each activity and each subject.

## Identifiers

* **Activity**: Type of Activity
* **Subject**: Subject ID

## Types of Activities

1. WALKING
1. WALKING_UPSTAIRS
1. WALKING_DOWNSTAIRS
1. SITTING
1. STANDING
1. LAYING
